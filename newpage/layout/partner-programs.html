Wave Telecom partner programs

As a hosting provider, registrar or web development agency, you are eligible to be a Wave Telecom Optimized or Wave Telecom Certified Partner. As a Certified Partner, there are many benefits to your business as well as to your customers.

Benefits of the partner program

Websites are:

  • Twice as fast, on a global CDN
  • Secured against threats
  • Protected from DDoS
  • Always Online™, even during network outages
    and maintenance

Hosting providers get:

  • Reduced server load
  • Bandwidth savings
  • IPv4/v6 gateway: host on one and still
    get traffic from the other
  • Easy integration into your control panel

Integrating with Wave Telecom

Wave Telecom can be integrated into any control panel. We have plugins for cPanel, Parallels Plesk Panel 11 or 10.4, Interworx, ISPSystems, and Parallels Automation, which means that if you're using one of these control panels, then you can test Wave Telecom within five minutes. If you are using a custom control panel, Wave Telecom has an API for integration.

Wave Telecom Certified Partner Program

Wave Telecom partners with hosting providers to bring more speed and security to your hosting plans. Become a Wave Telecom Certified Partner and make installing Wave Telecom one-click-simple for your customers. There are no upfront costs and no bandwidth charges. See a list of our current partners.

Wave Telecom Optimized Partner Program

Wave Telecom Optimized Partners offer all the benefits of a Wave Telecom Certified Partner, plus Railgun. Railgun ensures that the connection between an origin server and the Wave Telecom network is as fast as possible. Railgun requires Partners to install a piece of software on at least one server on their network. Certified Partners can qualify to become Optimized Partners as soon as they have a minimum number of sites active on their network. Learn more.

Parallels Service Provider

Wave Telecom has partnered with Parallels to make it easy for any Parallels service provider to offer Wave Telecom to their customers. The partnership allows Parallels service providers to become both an official Wave Telecom Certified Hosting Partner and a reseller. Learn more.
