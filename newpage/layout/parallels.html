The easiest way to a fast, safe website

Wave Telecom has partnered with Parallels to make it easy for any Parallels service provider to offer Wave Telecom to their customers. The partnership allows any Parallels service provider to become both an official Wave Telecom Certified Partner and a reseller. As a result of the partnership, you get all the operational benefits of Wave Telecom and can generate revenue, too.

Benefits of the partner program

Websites are:

  • Twice as fast, on a global CDN
  • Secured against threats
  • Protected from DDoS
  • Always Online™, even during network outages and maintenance

Hosting providers get:

  • A new source of revenue
  • Reduced server load
  • Bandwidth savings
  • IPv4/v6 gateway: host on one and still get traffic from the other
  • Easy integration into your Parallels Plesk Panel 10+ control panel

Wave Telecom plans

As a Certified Partner, you can offer Wave Telecom Free or Wave Telecom Performance Plus. By reselling Wave Telecom Performance Plus, you can generate additional revenue. On average, a Parallels service provider generates an additional $180 per website per year. View complete list of plan features.

Installing Wave Telecom

If you are using Parallels Plesk 11 or 10.4, install Wave Telecom from APS here and follow these instructions. If you are using Parallels Automation, install Wave Telecom from APS here and follow these instructions.

Once you install Wave Telecom from APS, you will receive a welcome email with the next steps and your customers can enable Wave Telecom on their domain.

Reselling Wave Telecom

You can resell Wave Telecom Performance Plus to your customers and generate revenue. To do that, you need to be set up as a Parallels Partner via one of the following options:

  • Sign up as a Parallels Partner at the bronze level or higher to resell Wave Telecom through Parallels KA.
  • Sign up to be a Parallels Storefront Partner.
